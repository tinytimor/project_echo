# Data Directory

This folder holds the **EchoNet-Pediatric** dataset and pre-extracted embeddings.
These files are too large for git (~2.6 GB) and must be downloaded separately.

---

## EchoNet-Pediatric Dataset

**7,810 labeled pediatric echocardiogram videos** from Stanford AIMI /
Lucile Packard Children's Hospital.

### Download Steps

1. Visit **https://stanfordaimi.azurewebsites.net/**
2. Search for **"EchoNet-Pediatric"**
3. Sign in with a **Microsoft account**
4. Accept the **Stanford Research Use Agreement** (DUA)
5. Click **"Export Dataset"** → copy the Azure SAS URL
6. Download:

```bash
# Option 1: azcopy (recommended — handles Azure blob storage natively)
# Install: https://learn.microsoft.com/en-us/azure/storage/common/storage-use-azcopy-v10
azcopy copy "<SAS_URL>" ./echonet_pediatric --recursive

# Option 2: wget (if SAS URL points to a zip)
wget -O echonet_pediatric.zip "<SAS_URL>"
unzip echonet_pediatric.zip -d ./echonet_pediatric
```

### Expected Structure

```
data/
├── echonet_pediatric/
│   ├── A4C/
│   │   ├── Videos/              (3,284 .avi files)
│   │   ├── FileList.csv         (FileName, EF, Sex, Age, Weight, Height, Split)
│   │   └── VolumeTracings.csv   (FileName, X, Y, Frame — LV contour points)
│   └── PSAX/
│       ├── Videos/              (4,526 .avi files)
│       ├── FileList.csv
│       └── VolumeTracings.csv
└── embeddings_videomae/          (auto-generated by train.sh)
    ├── pediatric_a4c/            (.pt embedding files + manifest.json)
    └── pediatric_psax/
```

### Dataset Details

| Property | Value |
|---|---|
| Total Videos | 7,810 |
| A4C Videos | 3,284 (EF 7.0%–73.0%, mean 60.9%) |
| PSAX Videos | 4,526 (EF 4.1%–73.0%, mean 61.3%) |
| Resolution | 112×112 pixels (de-identified) |
| Format | AVI |
| Ages | 0–18 years |
| Split | 10-fold CV (folds 0–7 train, 8 val, 9 test) |

### VideoMAE Embeddings

The `embeddings_videomae/` folder is **auto-generated** by `train.sh` during
Step 1 (embedding extraction). You do not need to download these — they are
created from the raw videos using `MCG-NJU/videomae-base` (downloaded
automatically from Hugging Face).

To regenerate:
```bash
bash train.sh --views A4C PSAX  # extracts embeddings as part of full pipeline
# or just embeddings:
python -m echoguard.regression.extract_videomae --views A4C PSAX
```
